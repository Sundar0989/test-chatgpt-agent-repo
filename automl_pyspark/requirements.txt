# ================================================
# TROUBLESHOOTING
# ================================================

# If you encounter issues with specific packages:
# 
# XGBoost issues:
#   - Ensure Java is installed and JAVA_HOME is set
#   - Try: pip install xgboost --no-cache-dir
#
# SynapseML (LightGBM) issues:
#   - Requires Scala 2.12 and Spark 3.2+
#   - May need: pip install synapseml[lightgbm]
#
# PySpark issues:
#   - Set PYSPARK_PYTHON environment variable
#   - For Windows: may need winutils.exe
#
# Memory issues:
#   - Increase JVM heap size: --driver-memory 4g --executor-memory 4g
